RMS Value (Sine Wave)

Irms = Imax / √2 = 0.707 Imax

  • Form Factor = RMS / Avg = 1.11
  • Peak Factor = Max / RMS = 1.414

Transformer EMF Equation

E = 4.44 × f × Φm × N

f is freq, Φm is max flux, N is turns.

AC Power Components

P = VI cosφ (Active, W)
Q = VI sinφ (Reactive, VAR)
S = VI (Apparent, VA)

Power Factor = cosφ = P / S = R / Z

Boolean Algebra Laws

A + AB = A (Absorption)
A + A'B = A + B
(A+B)' = A' · B' (DeMorgan's)
