01. Characteristic Equation

For a square matrix A, the equation obtained by setting the determinant of (A - λI) to zero is called the characteristic equation.

|A - λI| = 0

The roots of this equation are known as the Eigen Values (or characteristic roots) of matrix A.

02. Cayley-Hamilton Theorem

Statement:

"Every square matrix satisfies its own characteristic equation."

This theorem is extensively used in JNTUK exams to find the inverse of a matrix and higher powers of a matrix (e.g., A⁸ or A⁻¹).

03. Solving Step-by-Step

1

Formulate |A-λI|

Subtract λ from the principal diagonal elements of matrix A.

2

Expand Determinant

Expand the determinant to get a polynomial in λ (usually degree 3 for engineering apps).

3

Find Eigen Vectors

For each λᵢ, solve (A-λᵢI)X = 0 to find the corresponding vector X.

Academic Drills

Eigenvalue Concept Drill

Test your mastery of core matrix properties, transformations, and characteristic proofs using these exam-calibrated questions.

01

If a 3x3 matrix A has principal diagonal cells 2, 5, and -1, what represents the sum of its eigenvalues?

Core Property: Matrix Trace Link

Correct Answer: B (6) Trace Property: The sum of eigenvalues of a square matrix is always equal to its Trace (the sum of elements on its principal diagonal). Here, $Tr(A) = 2 + 5 + (-1) = 6$.
02

The eigenvalues of a 3x3 matrix are 1, -2, and 3. What is the determinant $|A|$ of this matrix?

Core Property: Product-Determinant Rule

Correct Answer: C (-6) Determinant Property: The product of the eigenvalues is equal to the determinant $|A|$ of the matrix. For eigenvalues $1, -2, 3$, the determinant evaluates to $(1) \times (-2) \times (3) = -6$.
03

According to the Cayley-Hamilton Theorem, if a matrix A has characteristic polynomial λ² - 3λ + 2 = 0, which matrix equation holds?

Core Property: Operator Polynomial Translation

Correct Answer: B (A² - 3A + 2I = O) Cayley-Hamilton: Every square matrix satisfies its own characteristic equation. When substituting the scalar variable $\lambda$ with the matrix $A$, you must multiply all constant coefficients by the identity matrix $I$ (yielding $+ 2I$), and replace $0$ with the null matrix $O$.
04

If the eigenvalues of a square matrix A are 2 and 5, what are the eigenvalues of its transpose Aᵀ?

Core Property: Transpose Invariance

Correct Answer: C (2 and 5) Transpose Rule: A square matrix $A$ and its transpose $A^T$ have identical characteristic determinants: $|A - \lambda I| = |A^T - \lambda I| = 0$. Therefore, transposing a matrix preserves all eigenvalues perfectly.
05

If λ is a non-zero eigenvalue of an invertible matrix A, what is the corresponding eigenvalue of its inverse A⁻¹?

Core Property: Spectral Reciprocals

Correct Answer: D (1/λ) Inverse Theorem: Since $AX = \lambda X$, multiplying by $A^{-1}$ on the left hand yields $X = \lambda A^{-1} X$. Since $\lambda \neq 0$, dividing yields $A^{-1} X = \frac{1}{\lambda} X$. Thus, $1/\lambda$ constitutes the direct eigenvalue.
