Eccentricity Ratio ($e$)

  • $e < 1$ (Ellipse)
  • $e = 1$ (Parabola)
  • $e > 1$ (Hyperbola)

Line AB, 75mm long, has end A 15mm above HP, 20mm in front of VP. Inclined at 30° to HP, 45° to VP. Draw projections.

🃏 VIVA_FLIP_CARDS //

Difference between View and Projection?

Isometric View uses true dimensions. Isometric Projection uses reduced scale (0.815 of true length).

Define 'Representative Fraction'.

Ratio of drawing length to actual physical length. R.F = L_draw / L_actual.

🚨 1-DAY EXAM SURVIVAL PROTOCOL //

  • 1. Master Unit II Lines: Guaranteed 14M question. Locus rotation method is mandatory.
  • 2. Conic Construction: Practice General Method for Ellipse (Eccentricity). Very high recurring probability.
  • 3. Sectioning Hatching: Always use light parallel lines at 45° for section surfaces to score neatness marks.
  • 4. CAD Syntax: Memorize L, C, TR, F commands for the 5M theory bits.
  • 5. Pencil Hygiene: Use 2H for all initial logic lines. HB only for final object profile.
