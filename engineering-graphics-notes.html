Unit I — Detailed Notes

Geometrical Constructions, Curves & Scales

🔥 99% Repeat

1. Dimensioning & Line Conventions

As set by Bureau of Indian Standards (BIS: SP46), different pencils dictate drawing hierarchy:

  • HB Pencil: Used for title blocks, object lines, boundary outlines, and text dimensions.
  • 2H Pencil: Used for extremely light construction lines, centers, and projection links.
  • Dimension lines: Must sit at least 8mm outside the drawing with a thickness ratio of 1:3 for arrows (typically 3mm long, 1mm wide).

2. Conic Sections: Tracing Curves

Curves formed by cutting a right circular cone at various slant angles. The curve type is governed by mathematical eccentricity (e).

Ellipse (e < 1) Slanted cutting plane passing through all cone generators.
Parabola (e = 1) Cutting plane parallel to one of the end generators.
Hyperbola (e > 1) Plane parallel to cone vertical axis, forming two branches.

3. Scale Engineering Metrics

Calculates mapping sizes. Always convert metrics to base dimensions.
Length of Scale (LOS) = Representative Fraction (R.F) × Maximum Scale Length

Unit II — Detailed Notes

Orthographic Projections (Points, Lines, Planes)

⚡ Easy 8 Marks

1. Orthographic Quadrants Breakdown

JNTUK strictly utilizes First Angle Projection rules:

• Object placed between observer and recording planes (HP / VP).
• Front View (a') projects ABOVE the horizontal fold line XY.
• Top View (a) projects BELOW the horizontal line XY.
• Left Side View projects on Right Profile plane. Right Side on Left Profile Plane.

2. Projections of Straight Lines inclined to both HP & VP

This is the most critical question in Unit II.

Step-by-step Tracing Protocol:
  1. Define direct coordinates of point A above HP/VP, mark top view a and front view a'.
  2. Draw local horizontal passing reference lines (Locus).
  3. Rotate true length TL with slant angles θ (slant to HP) and φ (slant to VP).
  4. Project end vertical intersection grids onto adjacent locus bands to retrieve real apparent views.
Unit III — Detailed Notes

Projections of Solids (Prisms, Pyramids, Revolution)

📘 Drawing Base

1. Polyhedral Classification vs Revolution Solids

Under standing solids criteria, JNTUK breaks exams into two families:

  • Polyhedras: Solids bound by flat planar faces (e.g., prisms having top/bottom regular bases, pyramids tapering upward to a single apex vertex).
  • Revolution Solids: Cylinders and right circular cones formed by rotating rectangles/triangles along a central core axis.

2. Stage Rotation Method:

To draw solid projection inclined to planes:
State 1: Draw base true shape on HP (for vertical slant angles) or VP (for horizontal slant angles).
State 2: Erect projections. Rotate front height boundary rectangles or triangles to match specified slant slope theta.
State 3: Draw linear linking grids down to map intersections. Render final apparent 2D projection shape. Ensure hidden bounding borders are accurately detailed as dotted lines.

Unit IV — Detailed Notes

Sections & Development of Surfaces

🔥 99% Repeat

1. Sectional Views & Extraction of True Shape

True shape must be projected by drawing a new auxiliary reference line parallel to direct tilt cutting planes, then drawing perpendicular grid links to retrieve undistorted surfaces.

Hatching Protocol: The cut slice must be detailed with light parallel lines spaced at identical 2mm increments inclined at exactly 45° to direct axes using a thin 2H pencil.

2. Development Techniques (Surface unfolding nets)

Parallel Line Development Used for prisms and solid cylinders. Base parameter is the unfolded perimeter length = π × d or base sum of polygon edges.
Radial Line Development Used for right circular cones and regular pyramids. Perimeter is traced by drawing arcs with sectoral sweep slant angles: θ = (Radius / Slant Height) × 360°
Unit V — Detailed Notes

Isometric Transformations & Computer CAD Graphics

🧠 Viva Important

1. Isometric View vs Isometric Projection

Avoid this classic mistake on exam day:

  • Isometric View: Drawn using actual physical dimensions. Isometric dimensions match real scale numbers directly.
  • Isometric Projection: Drawn using downscaled sizes to reflect foreshortened visual effects. Scaled length equals exactly 0.816 × True Length.

QUESTION 1

Why are sectional views used in design?

Hover to Flip Cards
ANSWER

To reveal hidden internal features of a complex object that are unclear with hidden (dotted) lines alone.

QUESTION 2

Define the core difference between Isometric View & Projection.

Hover to Flip Cards
ANSWER

Isometric View uses true linear dimensions. Isometric Projection downscales lengths by a factor of 0.816.

QUESTION 3

What is an involute?

Hover to Flip Cards
ANSWER

An spiral path traced by a point on a tight string as it winding/unwinding around a base polygon or circle.

QUESTION 4

State the sweep angle calculation parameters for cone unfolding.

Hover to Flip Cards
ANSWER

Sector Angle (θ) = (Base Radius ÷ slant height generator) × 360 degrees.
