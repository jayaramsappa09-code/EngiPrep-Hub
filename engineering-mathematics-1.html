Eigenvalues & Trans.

Cayley-Hamilton, Diagonalization

▶ Cayley-Hamilton Theorem

Every square matrix satisfies its own characteristic equation. If |A - λI| = 0 gives λ³ - c₁λ² + c₂λ - c₃ = 0, then A³ - c₁A² + c₂A - c₃I = 0.

Uses: Finding A⁻¹ and high powers of A easily.

Quadratic Form Rules

  • Positive Definite: All λ > 0
  • Negative Definite: All λ < 0
  • Pos. Semidefinite: All λ ≥ 0 (at least one 0)
  • Indefinite: Mix of +ve and -ve λ

Diagonalization Process (D = P⁻¹AP)

1. Find λ
→
2. Find Eigenvectors (X)
→
3. Form P = [X1 X2 X3]
→
4. Target: D = diag(λ)

Calculus

Mean Value Theorems, Taylor Series

Rolle's Theorem

  1. Continuous in [a,b]
  2. Differentiable in (a,b)
  3. f(a) = f(b)

Then ∃ c ∈ (a,b) where f'(c) = 0

Lagrange MVT

  1. Continuous in [a,b]
  2. Differentiable in (a,b)

f'(c) = [f(b) - f(a)] / [b - a]

Cauchy MVT

Two functions f(x), g(x) satisfying continuity/diff conditions.

f'(c)/g'(c) = [f(b)-f(a)] / [g(b)-g(a)]

Series Expansions Vault

Taylor (at x=a): f(x) = f(a) + (x-a)f'(a) + (x-a)²f''(a)/2! + ...
Maclaurin (Taylor at x=0): f(x) = f(0) + xf'(0) + x²f''(0)/2! + ...

Partial Differentiation

Jacobians, Maxima & Minima

Jacobian (J)

If u=f(x,y) and v=g(x,y), then J is the determinant of partial derivatives.

J(u,v)/J(x,y) =
| ∂u/∂x ∂u/∂y |
| ∂v/∂x ∂v/∂y |

* If J = 0, functions are functionally dependent.

Total Derivative

If u = f(x,y) and x=φ(t), y=ψ(t) (Chain Rule)

du/dt = (∂u/∂x)(dx/dt) + (∂u/∂y)(dy/dt)

Maxima & Minima Flow (2 Variables)

1. Find stationary points: Solve ∂f/∂x = 0 & ∂f/∂y = 0

2. Calculate 2nd derivatives at point (a,b):

r = ∂²f/∂x², s = ∂²f/∂x∂y, t = ∂²f/∂y²

3. Evaluate rt - s²:

  • If rt-s² > 0 and r < 0 ➔ Maximum
  • If rt-s² > 0 and r > 0 ➔ Minimum
  • If rt-s² < 0 ➔ Saddle Point
  • If rt-s² = 0 ➔ Further calc needed

Multiple Integrals

Order Change, Area, Volume

Coordinate Transform (2D)

Cartesian (x,y) to Polar (r,θ):

x = r cos(θ)

y = r sin(θ)

dx dy = r dr dθ

Applications

  • Area: A = ∬ dx dy
  • Volume (Double): V = ∬ z dx dy
  • Volume (Triple): V = ∭ dx dy dz

Change of Order of Integration

If limits of an integral ∬ f(x,y) dx dy are complex, changing the order to dy dx can simplify it. Crucial steps: 1. Extract given limits & sketch region. 2. Identify new boundary strips (horizontal to vertical or vice versa). 3. Find new limit equations from the sketch.

What is a Singular Matrix?
A square matrix whose determinant is Zero (|A| = 0). Inverse doesn't exist.
Sum of Eigenvalues = ?
Trace of the matrix (sum of principal diagonal elements). Product = Determinant.
Geometrical meaning of Rolle's?
There is at least one point where the tangent is parallel to the x-axis.
What is an Orthogonal Matrix?
A matrix A where A * Aᵀ = I (Identity). Also Aᵀ = A⁻¹.
