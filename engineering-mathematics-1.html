Rank of a Matrix 🔥 Core

💡
The rank of matrix A (ρ(A)) is the order of the largest non-zero minor — or equivalently, the number of non-zero rows after reducing to Row Echelon Form.
Key Properties
Rank of a Matrix — ρ(A)
Definition
ρ(A) = number of non-zero rows in Row Echelon Form
  • ρ(A) ≤ min(m,n) for an m×n matrix
  • ρ(A) = n iff A is non-singular (square)
  • ρ(Aᵀ) = ρ(A) — transpose preserves rank
  • ρ(AB) ≤ min(ρ(A), ρ(B))
🎯 Matrix Rank Visualizer LIVE

Click any cell to edit. Press Find Rank to animate row reduction.

Simple: Each row is a "direction" in space. Rank counts how many rows point in genuinely different directions. If one row is a multiple of another — it's redundant. Rank = number of truly independent directions.
Intuition: Imagine 3 arrows in 3D space. If all lie in the same plane, you only have 2 independent directions — rank=2. Rank measures the "dimensionality" of the space spanned by the rows.
Exam Shortcut: Reduce to REF → count non-zero rows. For 3×3: if det≠0 → rank 3. If det=0 but some 2×2 minor≠0 → rank 2. All 2×2 minors=0 but some entry≠0 → rank 1.
Mistakes: (1) Forgetting rank ≤ min(rows,cols). (2) Confusing "all rows non-zero" with "linearly independent." (3) Not reducing fully — always go to complete REF.

Normal Form (Canonical Form)

📐
Every m×n matrix A of rank r can be reduced by row AND column operations to the Normal Form: a matrix with Iᵣ in top-left, zeros elsewhere.
Normal Form
PAQ = ⌈ Iᵣ   0 ⌉
  ⌊ 0    0 ⌋
🎯
Key Use: The normal form directly reveals the rank r of A — it's the size of the identity block. Used to verify rank and to prove theoretical results.

Cauchy-Binet Formula

Theorem
Cauchy-Binet Formula
For A (m×n) and B (n×m), m ≤ n
det(AB) = Σ det(A[S]) · det(B[S])
sum over all m-element subsets S of {1,...,n}
  • Special case m=n: det(AB) = det(A)·det(B)
  • If m > n: det(AB) = 0
  • Generalizes the product rule for determinants
📝
Exam Tip: For JNTUK, typically only the special case det(AB) = det(A)·det(B) is tested. The full Cauchy-Binet is a theoretical result — understand the statement and the m > n case.

Inverse by Gauss-Jordan Method 🔥 Exam Fav

💡
Idea: Form [A | I]. Apply row operations to reduce A → I. The right block transforms to A⁻¹.
Method
[A | I] →(row operations)→ [I | A⁻¹]

Homogeneous Linear Equations

Theory
System AX = 0
  • Always has the trivial solution X = 0
  • Non-trivial solution ⟺ ρ(A) < n (unknowns)
  • Number of independent solutions = n − ρ(A)
  • det(A) = 0 is necessary for non-trivial solution
Nullity
Nullity = n − ρ(A) = dimension of solution space

Non-Homogeneous Equations 🔥 Core

Rouché–Capelli Theorem
Consistency of AX = b
  • Consistent ⟺ ρ(A) = ρ([A|b])
  • ρ(A) = ρ([A|b]) = n → Unique solution
  • ρ(A) = ρ([A|b]) < n → Infinite solutions (n−ρ free vars)
  • ρ(A) ≠ ρ([A|b]) → No solution (inconsistent)
Quick Decision
Compare ρ(A) with ρ([A|b]) with n

Eigenvalues & Eigenvectors 🔥 Core

💡
AX = λX — the eigenvector X doesn't change direction under A; it only scales by λ. These are the "natural axes" of the transformation.
Definition
Eigenvalue Equation
AX = λX  ⟺  (A − λI)X = 0
  • Non-trivial solution exists iff det(A−λI) = 0
  • This equation gives the characteristic polynomial
  • Roots of characteristic polynomial = eigenvalues
  • Sum λᵢ = trace(A); Product λᵢ = det(A)

Simple: Push a rubber sheet with a matrix. Most vectors rotate AND stretch. Eigenvectors only stretch (or flip) — like a rubber band along a fixed axis that keeps its direction.
Intuition: λ>1: vector stretches. 0<λ<1: shrinks. λ<0: flips. These special directions are where the matrix is "most natural." All complex dynamics decompose along these axes.
Shortcut: 2×2: λ² − tr(A)λ + det(A) = 0. Always verify: Σλᵢ = trace, Πλᵢ = det. For symmetric matrix, eigenvectors for distinct eigenvalues are orthogonal!

Cayley-Hamilton Theorem 🔥 Exam Fav

Theorem
Cayley-Hamilton

Every square matrix satisfies its own characteristic equation.

If p(λ) = det(A−λI), then p(A) = 0
  • 2×2: A² − tr(A)·A + det(A)·I = 0
  • Used to find A⁻¹ and Aⁿ without row operations
  • A⁻¹ = [tr(A)·I − A] / det(A) for 2×2
✅ Cayley-Hamilton Verifier VERIFY

Using C-H to Find A⁻¹ — Step by Step

1A = [[2,1],[1,2]]. Find char. poly: p(λ) = λ²−4λ+3
trace=4, det=3 → p(λ)=λ²−4λ+3
2By C-H: A²−4A+3I = 0
A²=[[5,4],[4,5]], 4A=[[8,4],[4,8]], 3I=[[3,0],[0,3]]
A²−4A+3I = [[0,0],[0,0]] ✓
3Rearrange to isolate A⁻¹
A(A−4I) = −3I → A⁻¹ = −(1/3)(A−4I)
4Compute A⁻¹
A⁻¹ = (1/3)[[2,−1],[−1,2]]
Verify: A·A⁻¹ = I ✅
No row operations needed — pure algebra via C-H.

Diagonalization ⭐ Important

💡
A = PDP⁻¹ where D is diagonal (eigenvalues) and P's columns are eigenvectors. Then Aⁿ = PDⁿP⁻¹ — computing A¹⁰⁰ becomes trivial!
Diagonalization
D = P⁻¹AP  |  Columns of P = eigenvectors  |  Diagonal of D = eigenvalues

Complete Diagonalization Procedure

1Solve det(A−λI)=0 → eigenvalues λ₁,λ₂,…
2For each λᵢ, solve (A−λᵢI)X=0 → eigenvectors Xᵢ
3Form P = [X₁|X₂|X₃] (columns = eigenvectors)
4Form D = diag(λ₁,λ₂,λ₃)
D = [[λ₁,0,0],[0,λ₂,0],[0,0,λ₃]]
5Verify PDP⁻¹ = A  |  Use Aⁿ = PDⁿP⁻¹
⚡
Dⁿ is trivial: just raise each diagonal entry to nth power!

Classification

Rolle's Theorem ⭐ Core

Theorem
Rolle's Theorem

If f is continuous on [a,b], differentiable on (a,b), and f(a) = f(b), then ∃ c ∈ (a,b) such that f'(c) = 0.

Conclusion
∃ c ∈ (a,b) : f'(c) = 0   (horizontal tangent exists)
  • Geometrically: the curve must have at least one horizontal tangent
  • Special case of Lagrange MVT with f(a) = f(b)
  • Conditions: continuity on [a,b], differentiability on (a,b), f(a) = f(b)
📈 Rolle's Theorem Visualizer ANIMATE

f(−1)=0 = f(1)=0 ✓  |  f'(x)=3x²−1=0 → x=±1/√3 ≈ ±0.577  |  Rolle's Theorem confirmed ✓

Lagrange Mean Value Theorem 🔥 Core

Theorem
Lagrange MVT (LMVT)

If f is continuous on [a,b] and differentiable on (a,b), then:

LMVT Formula
∃ c ∈ (a,b) : f'(c) = [f(b) − f(a)] / (b − a)
  • The instantaneous rate equals the average rate at some point c
  • Geometrically: tangent at c is parallel to the chord AB
  • Generalizes Rolle's Theorem (when f(a)=f(b), slope = 0)

Average slope = (9−1)/(3−1) = 4  |  f'(c) = 2c = 4 → c = 2  |  c=2 ∈ (1,3) ✓

Cauchy Mean Value Theorem

Theorem
Cauchy MVT

If f and g are continuous on [a,b], differentiable on (a,b), and g'(x) ≠ 0, then:

∃ c ∈ (a,b) : f'(c)/g'(c) = [f(b)−f(a)] / [g(b)−g(a)]
  • Generalizes LMVT (take g(x)=x)
  • Used to prove L'Hôpital's rule
  • Parametric interpretation: tangent parallel to chord in parametric curve (g(t),f(t))

Partial Derivatives 🔥 Core

💡
∂f/∂x: differentiate with respect to x, treating all other variables as constants. It gives the slope of the surface in the x-direction.
Definition
∂f/∂x = lim[h→0] [f(x+h,y) − f(x,y)] / h
Important Results
Higher-Order Partial Derivatives
  • ∂²f/∂x² : second derivative w.r.t. x (concavity in x)
  • ∂²f/∂x∂y : mixed partial (first x then y)
  • Clairaut's Theorem: ∂²f/∂x∂y = ∂²f/∂y∂x (if continuous)
  • For f(x,y)=xⁿyᵐ: ∂f/∂x = nxⁿ⁻¹yᵐ, ∂f/∂y = mxⁿyᵐ⁻¹

Total Derivative

Total Differential
df = (∂f/∂x)dx + (∂f/∂y)dy + (∂f/∂z)dz
💡
If x = x(t), y = y(t), the total derivative is: df/dt = (∂f/∂x)(dx/dt) + (∂f/∂y)(dy/dt)

Total Derivative — Worked Example

1Given u = x²y, x = t², y = t³. Find du/dt.
2Compute partial derivatives
∂u/∂x = 2xy  |  ∂u/∂y = x²
3Apply total derivative formula
du/dt = (2xy)(2t) + (x²)(3t²)
4Substitute x=t², y=t³
= 2(t²)(t³)(2t) + (t⁴)(3t²) = 4t⁶ + 3t⁶ = 7t⁶ ✅
Verify: u = x²y = t⁴·t³ = t⁷. du/dt = 7t⁶ ✓ Direct match!

Chain Rule for Partial Derivatives

Chain Rule — if u=f(x,y), x=x(r,s), y=y(r,s)
∂u/∂r = (∂u/∂x)(∂x/∂r) + (∂u/∂y)(∂y/∂r)
🔗
Tree Method: Draw a dependency tree from u to the final variables. Each path from u to a variable contributes one term; multiply along each path and sum all paths.

Jacobians 🔥 Exam Fav

Definition
Jacobian of Transformation
Jacobian J = ∂(u,v)/∂(x,y)
J = det | ∂u/∂x   ∂u/∂y |
       | ∂v/∂x   ∂v/∂y |
  • J ≠ 0 is needed for a valid transformation (inverse exists)
  • In integration: ∬f(u,v) du dv = ∬f(x,y)|J| dx dy
  • If J(u,v/x,y) · J(x,y/u,v) = 1 (inverse Jacobians)
  • Used to check functional dependence: if J=0 → functions are dependent

Maxima & Minima of f(x,y) 🔥 Core

Second-Order Test
Critical Points Classification

At a critical point (a,b) where fₓ=0 and fᵧ=0, compute:

Discriminant
D = fₓₓfᵧᵧ − (fₓᵧ)²
  • D > 0, fₓₓ > 0 → Local Minimum
  • D > 0, fₓₓ < 0 → Local Maximum
  • D < 0 → Saddle Point
  • D = 0 → Test inconclusive

Lagrange Multipliers ⭐ Important

💡
Optimize f(x,y,z) subject to constraint g(x,y,z)=0 by solving ∇f = λ∇g. The multiplier λ has useful interpretations (sensitivity of optimal value to constraint).
System to Solve
fₓ = λgₓ  |  fᵧ = λgᵧ  |  f_z = λg_z  |  g(x,y,z) = 0

Lagrange — Classic Example: Max of x+y+z on x²+y²+z²=1

1Set up: f=x+y+z, g=x²+y²+z²−1
2∇f = λ∇g → (1,1,1) = λ(2x,2y,2z)
1=2λx → x=1/(2λ)
1=2λy → y=1/(2λ)
1=2λz → z=1/(2λ)
3Use constraint: x²+y²+z²=1
3/(4λ²) = 1 → λ = ±√3/2
x=y=z=1/√3 (for max)
4Maximum = x+y+z = 3/√3 = √3 ✅
By symmetry, the maximum on a unit sphere of a linear function is along the direction of the gradient.

Double Integrals 🔥 Core

💡
A double integral ∬f(x,y) dA gives the volume under the surface z=f(x,y) over the region R. Computed as iterated (repeated) integrals.
Iterated Integral
∬_R f(x,y) dA = ∫_a^b [∫_{g₁(x)}^{g₂(x)} f(x,y) dy] dx
Evaluation Steps
Computing Double Integrals
  • Step 1: Identify the region R (sketch it!)
  • Step 2: Set up limits — inner integral over y, outer over x (or vice versa)
  • Step 3: Evaluate inner integral (treat x as constant)
  • Step 4: Evaluate outer integral over the result
🗺️ Double Integral Region Visualizer VISUAL

Region: 0 ≤ x ≤ 2, 0 ≤ y ≤ x. Shaded = integration region.

∫₀² ∫₀ˣ (x+y) dy dx = ∫₀² [xy + y²/2]₀ˣ dx = ∫₀² (3x²/2) dx = [x³/2]₀² = 4

Step-by-Step: Evaluate ∫₀¹ ∫₀¹ (x²+y²) dx dy

1Inner integral: ∫₀¹ (x²+y²) dx (treat y as constant)
2= [x³/3 + y²x]₀¹ = 1/3 + y²
3Outer integral: ∫₀¹ (1/3 + y²) dy
= [y/3 + y³/3]₀¹ = 1/3 + 1/3 = 2/3
4Answer: 2/3 ✅
By symmetry, ∫∫(x²+y²)dA over [0,1]² = 2·∫∫x² dA = 2·(1/3) = 2/3.

Change of Order of Integration 🔥 Exam Fav

💡
Key idea: Sometimes switching the order of integration simplifies computation. Sketch the region, identify the same area with reversed limits.

Change of Order: ∫₀¹ ∫_x¹ f(x,y) dy dx

1Identify region: 0≤x≤1, x≤y≤1
✏️
Region is the triangle above y=x and below y=1, with 0≤x≤1.
2Sketch: x from 0 to 1, y from x to 1
📐
Same triangle, now described as: y from 0 to 1, and for each fixed y, x from 0 to y.
3New limits after changing order:
∫₀¹ ∫_x¹ f dy dx = ∫₀¹ ∫₀^y f dx dy
4Apply to ∫₀¹ ∫_x¹ e^(y²) dy dx
= ∫₀¹ ∫₀^y e^(y²) dx dy = ∫₀¹ y·e^(y²) dy = [e^(y²)/2]₀¹ = (e−1)/2 ✅
The original order had no closed form for ∫e^(y²)dy. Changing order makes it solvable!

Polar Coordinates ⭐ Important

Transformation to Polar
x = r cosθ, y = r sinθ  |  dA = r dr dθ  |  x²+y² = r²
⚡
The extra factor r in dA = r dr dθ is crucial — it comes from the Jacobian of the polar transformation: J = r.

Triple Integrals

Triple Integral
∭_V f(x,y,z) dV = ∫∫∫ f(x,y,z) dx dy dz
Coordinate Systems
Volume Elements
  • Cartesian: dV = dx dy dz
  • Cylindrical: x=r cosθ, y=r sinθ, z=z → dV = r dr dθ dz
  • Spherical: x=ρsinφcosθ, y=ρsinφsinθ, z=ρcosφ → dV = ρ²sinφ dρ dφ dθ
Choose coordinates based on symmetry of region
Spherical for spheres | Cylindrical for cones, cylinders | Cartesian for boxes

Areas & Volumes by Integration 🔥 Exam Fav

Area & Volume Formulas
Area = ∬_R dA  |  Volume = ∭_V dV
Vol. of Sphere (r=a): (4/3)πa³ via ∭dV in spherical
Area under curve: ∫∫ dy dx

Volume of Sphere using Triple Integral

1Sphere: x²+y²+z²=a². Use spherical coordinates.
2Limits: 0≤ρ≤a, 0≤φ≤π, 0≤θ≤2π
3V = ∫₀^{2π} ∫₀^π ∫₀^a ρ²sinφ dρ dφ dθ
= [∫₀^{2π}dθ][∫₀^π sinφ dφ][∫₀^a ρ² dρ]
= 2π · 2 · a³/3
4V = (4/3)πa³ ✅
The separability of the integral into three independent parts is a key advantage of spherical coordinates here.
