📏
Lines, Lettering & Dimensioning
IMP
▼
📐 Why This Drawing Matters
Real-World Connection
Every blueprint, manufacturing drawing, and architectural plan uses standardized lines and lettering. Without this, a 5mm thick wall looks the same as a 500mm wall. Engineers speak through lines — you must speak their language.
Line Types
  • Continuous thick — Visible outlines (0.7mm)
  • Continuous thin — Dimensions, hatching (0.35mm)
  • Dashed thin — Hidden edges (0.35mm)
  • Chain thin — Center lines, axis
  • Chain thick — Cutting planes
Lettering Rules
  • Height ratio — 7:10 (width:height)
  • Single stroke — Gothic style preferred
  • Vertical or 75° inclined
  • Min height — 3.5mm for notes
  • Min height — 5mm for dimensions
  • No serifs — clean block letters
✏️ Dimensioning Rules
1
Extension Lines
Draw thin lines extending 2–3mm beyond dimension line. Leave a 1mm gap from the object outline. Never cross dimension lines.
2
Dimension Lines
Place parallel to the feature, between extension lines. Use arrowheads at both ends (3:1 ratio). Keep 8–10mm from object.
3
Leader Lines
Thin inclined line ending in arrowhead, used for circles, radii, and notes. Always end at center of feature.
4
Units & Notation
All dimensions in mm (no unit symbol needed). Angles in degrees. Write ⌀ for diameter, R for radius.
⚠️ Common Drawing Mistakes
✕
Drawing extension lines touching the object — leave a 1mm gap
✕
Crossing dimension lines — plan layout before drawing
✕
Writing dimensions inside tiny spaces — use leader lines instead
✕
Inconsistent letter heights — use guidelines always
⚡ Quick Revision
Line Thickness

Thick = 0.7mm (visible), Thin = 0.35mm (rest), Ratio = 2:1

Lettering

Single stroke Gothic, 75° incline or vertical, 3.5mm min height

Geometrical Constructions & Regular Polygons
HOT
▼
🧠 Visual Imagination
Spatial Thinking Mode

A regular polygon is like a perfectly fair shape — every side equal, every angle equal. Imagine slicing a circle into equal pieces like a pizza. Connect those cut points — that's your polygon! The more slices, the more it looks like a circle.

✏️ Step-by-Step: Pentagon (5 sides) given side length
1
Draw Base
Draw AB = given side length. Mark midpoint M of AB.
2
Erect Perpendicular
At M, erect a perpendicular bisector MP of AB. Mark P4 on this line such that P4 is above M by length AB/2.
3
Locate Centers
Mark P6 (for hexagon) at a distance AB from M. Midpoint of P4 and P6 gives P5 (pentagon center). Similarly locate centers for other polygons.
4
Draw Circle & Mark Vertices
With P5 as center, radius P5A, draw circumscribed circle. Divide into 5 equal parts using compass. Connect to get pentagon.
Interior Angles
  • Triangle (3) — 60°
  • Square (4) — 90°
  • Pentagon (5) — 108°
  • Hexagon (6) — 120°
  • Heptagon (7) — 128.57°
  • Octagon (8) — 135°
Interior Angle Formula
Interior angle = (n-2) × 180° / n

Each central angle = 360° / n
🚀 Shortcut for Hexagon
For a regular hexagon: the circumscribed circle radius = side length. So draw a circle, step off the radius 6 times around it — done!
⚡ Quick Revision
Key Formula

Interior angle = (n–2)×180/n. Hexagon: radius = side length.

Method

General polygon method uses perpendicular bisector of base side with scaled center points P4 through Pn.

Conic Sections — Ellipse, Parabola & Hyperbola
HOT
▼
📐 Why This Drawing Matters
Ellipse Applications
  • Planetary orbits around the sun
  • Isometric circles in drafting
  • Arches in bridges
  • Gear design
Parabola & Hyperbola
  • Parabola → satellite dish, headlights
  • Parabola → projectile path
  • Hyperbola → nuclear cooling towers
  • Hyperbola → aircraft wing profiles
🧠 Conic Section Concept
Cone Slicing Visualization

Imagine a double ice-cream cone. Now slice it with a flat plane at different angles. Horizontal cut = Circle. Slight tilt = Ellipse. Parallel to cone side = Parabola. Steeper than cone side = Hyperbola. Every conic is just a cone being sliced!

Eccentricity e: Circle=0, Ellipse <1, Parabola=1, Hyperbola >1
✏️ Ellipse — Concentric Circle Method
1
Draw Axes
Draw major axis AB and minor axis CD intersecting at center O. Major axis is the longer one.
2
Draw Two Circles
Draw two concentric circles: one with radius OA (major) and one with radius OC (minor).
3
Radial Lines
Draw radial lines from O at equal angular intervals (12 or 16 lines). These cut both circles.
4
Project Points
From outer circle intersection, draw horizontal line. From inner circle intersection, draw vertical line. Their meeting point is on the ellipse.
5
Connect
Join all plotted points with a smooth freehand curve. Use French curve for precision.
✏️ Parabola — Rectangle Method
1
Draw Enclosing Rectangle
Draw rectangle with width = span, height = rise of parabola. Mark vertex V at bottom center.
2
Divide Top Edge & Side
Divide the top half-width into equal parts (say 4). Divide height into same number of equal parts.
3
Draw Lines
From vertex V, draw lines to points on top edge. From base divisions, draw verticals. Intersections = parabola points.
4
Smooth Curve
Connect all points with smooth curve through vertex V. Mirror for other half.
✏️ Hyperbola — Focus-Directrix Method (Eccentricity)
1
Set Up Axes
Draw directrix DD₁. Mark focus F at given distance. The vertex V lies between them at ratio e.
2
Locate Vertex
VF/VD = e. If e=3/2: divide VD into 2, VF into 3. Mark V at correct ratio from directrix.
3
Plot Points
Take points 1,2,3... on axis. For each: draw arc (center F, radius=e×distance to directrix). Intersection = curve point.
4
Join
Connect all plotted points with smooth curve. Draw both branches if needed.
🎯 Exam Formula Card
Ellipse: (x/a)² + (y/b)² = 1 | a=semi-major, b=semi-minor Parabola: y² = 4ax | focus at (a,0), directrix x = -a Hyperbola: (x/a)² - (y/b)² = 1 | e = c/a where c² = a²+b²
⚠️ Common Drawing Mistakes
✕
Confusing major and minor axes — major is always the longer axis
✕
Not using enough points — minimum 8 points for smooth ellipse
✕
Making parabola vertex too sharp — it must be smooth and rounded
⭐ PYQ Analysis
2023
Draw an ellipse with major axis 120mm and minor axis 80mm using concentric circle method. Draw tangent and normal at a point on the curve.
2022

2021
Construct a hyperbola with eccentricity 3/2, distance between directrix and focus = 60mm.

Special Curves — Cycloids & Involutes
HOT
▼
🧠 Visual Imagination
Mental Movie: Cycloid

Cycloid: Imagine a bicycle wheel rolling on a road. Put a glowing dot on the rim. As the wheel rolls (without slipping), that dot traces a beautiful wave — that is the cycloid. It's the path of a point on a rolling circle.

Involute: Imagine you tied a string to a pencil and wound the string tightly around a circular drum. Now, hold the drum fixed and slowly unwind the string while keeping it taut. The pencil traces the involute — it spirals outward from the circle.

Application: Gear teeth are involute curves — that's why gears mesh smoothly!

✏️ Cycloid Construction
1
Draw Base Line
Draw a horizontal base line of length = πD (circumference of rolling circle). D = given diameter.
2
Divide Circle & Line
Divide the rolling circle into 12 equal parts (number the divisions 0–12). Divide base line into 12 equal parts.
3
Find Centers
Draw horizontal line through center of circle (locus of center). Divide it into 12 equal parts → these are C₁, C₂... C₁₂.
4
Plot Points
With each Cₙ as center (radius = R), draw arcs. From each arc, mark the transferred point (using compass = distance from wheel division to original point). This gives cycloid points.
5
Join Smooth Curve
Connect all plotted points with a smooth curve. The curve starts and ends on the base line.
✏️ Involute of Circle Construction
1
Draw Circle
Draw a circle of given diameter. Divide it into 12 equal parts. Number divisions 0–12.
2
Draw Tangents
At each division point, draw a tangent to the circle. The tangent length at point n = n × (πD/12).
3
Mark Points
From division 1, measure (1/12)πD along its tangent. From division 2, measure (2/12)πD, and so on.
4
Connect
Join all endpoints smoothly. The curve starts at point 0 on circle and spirals outward.
Tangent & Normal Rule

Cycloid: Normal at any point passes through the corresponding point of contact on the base line. Tangent is perpendicular to normal.

Involute: Normal at any point is tangent to the base circle at that point. Tangent is perpendicular to this.

⭐ PYQ Analysis
2023
Draw a cycloid for a rolling circle of diameter 50mm. Draw tangent and normal at a point 30mm above the base line.
2022
Draw the involute of a circle of diameter 40mm. Draw tangent and normal at a point on the curve.
⚡ Quick Revision
Cycloid Memory

Rolling wheel → dot on rim → wave curve. Base line = πD. 12 divisions = 12 points.

Involute Memory

Unwinding string → spiral. Each tangent length = n×(πD/12). Used in gear tooth design.

Scales — Plain, Diagonal & Vernier
IMP
▼
📐 Why This Drawing Matters
Real-World Connection
Scales let engineers draw a real 10-meter bridge on an A1 sheet. Architects use 1:50 scales. Civil maps use 1:1000 or more. You're compressing or expanding reality into paper dimensions.
Representative Fraction (RF) = Drawing Length / Actual Length Scale = Drawing / Actual | If Scale = 1:50, RF = 1/50 Least Count of Plain Scale = 1 main division Least Count of Diagonal Scale = 1/(n divisions) of smallest main division
✏️ Plain Scale Construction
1
Calculate Drawing Length
Drawing length = RF × Max length to represent. Eg: RF=1/40, max=6m → Drawing length = (1/40)×6000mm = 150mm
2
Draw Base Line
Draw a horizontal line of calculated length. Divide into the required number of main divisions (each = 1 main unit).
3
Sub-divide First Division
Sub-divide the leftmost division into 10 equal parts (these are the sub-units). Mark zero at the junction.
4
Mark Dimensions
Number main divisions to the right of zero, sub-divisions to the left. Write RF and title at bottom.
✏️ Diagonal Scale (3 measurements)
1
Draw Plain Scale Rectangle
Construct the plain scale but make it a rectangle of height 10 units above the baseline.
2
Divide Height
Divide the height of the first sub-division into 10 equal horizontal lines.
3
Draw Diagonals
In first sub-division, draw diagonal from bottom-left of sub-div to top-right. Add parallel diagonals for each row.
4
Read Value
Any length can be read in 3 units: main, sub, and sub-sub divisions using the diagonal intersection.
⭐ PYQ Pattern
Always given: RF, maximum length to represent, and a specific measurement to mark on scale. Remember to calculate drawing length first, then construct.
⚡ Quick Revision
RF Formula

RF = Drawing/Actual. Plain scale = 2 measurements. Diagonal = 3 measurements. Vernier = 2 measurements with precision.

Key Steps

1. Calculate drawing length. 2. Draw & divide. 3. Sub-divide first division. 4. Add diagonals for diagonal scale.

Orthographic Projections
🔲
Introduction — Reference Planes & Quadrants
IMP
▼
🧠 Visual Imagination
The Glass Box Method

Imagine putting an object inside a transparent glass box. Now look at the object from the front → Front View (FV). Look from the top → Top View (TV). Look from the right side → Side View (SV). Now unfold this glass box flat — that's orthographic projection! Every engineering drawing is just an unfolded glass box.

⚙️ Reference Planes
Three Planes
  • HP — Horizontal Plane (looking from top)
  • VP — Vertical Plane (looking from front)
  • PP — Profile Plane (looking from side)
  • HP ∩ VP = XY line (reference line)
  • All projections are drawn on these planes
Four Quadrants
I

In front VP
II
Above HP
Behind VP
III
Below HP
Behind VP
IV
Below HP
In front VP
🎯 India / JNTUK uses First Angle Projection
In First Angle (1st quadrant): Object is BETWEEN observer and plane. FV on VP (front), TV on HP (below the XY line when unfolded). Side view goes LEFT of FV.
First Angle vs Third Angle

1st Angle: TV below XY, Side view to left. Plane is between object and observer.

3rd Angle: TV above XY, Side view to right. Observer is between object and plane. Used in USA.

Point Visualization

A point in space has 3 positions relative to HP and VP. Think of a glowing dot floating in a room. How high is it above the floor (HP)? How far from the front wall (VP)? Your answers directly tell you where to draw its projections.

1
Draw XY line
Draw horizontal reference line XY. This represents the intersection of HP and VP (the hinge line).
2

FV = projection on VP. Distance from XY = distance of point from HP (height). Above XY if above HP, below if below HP.

Projection of Straight Lines
HOT
▼
Line in Space — Visualize This

A line connects two points A and B in space. If the line tilts toward you (toward VP), its top view gets shorter. If it tilts downward (toward HP), its front view gets shorter. When it tilts both ways — you get apparent lengths in both views. The TRUE LENGTH is always longer than both apparent lengths.

Line PositionFVTV
Parallel to both HP & VPTrue length, parallel to XYTrue length, parallel to XY
Parallel to HP, inclined to VP at angle φShorter (apparent), parallel to XYTrue length, at angle φ to XY
Parallel to VP, inclined to HP at angle θTrue length, at angle θ to XYShorter (apparent), parallel to XY

🚀 True Length & Angles — Rotation Method
1
Draw Both Views
Draw FV (a'b') and TV (ab) with given data. Mark distances carefully from XY.
2
Rotate TV
In TV, rotate ab about a (keeping a fixed) until ab becomes parallel to XY. New position is ab₁.
3
Project Up
Project b₁ up to FV (draw vertical from b₁). From b', draw horizontal. Intersection gives b₁'. Length a'b₁' = TRUE LENGTH.
4
Find Angle θ
Angle θ (inclination with HP) = angle a'b₁' makes with XY. Angle φ (inclination with VP) = angle ab₁ makes with XY in TV after rotating FV.
⚠️ Key Rules
  • True length is ALWAYS greater than apparent length
  • θ = inclination with HP, found in FV after rotation
  • φ = inclination with VP, found in TV after rotation
  • Sum rule: α > θ and β > φ (apparent angles are larger)
  • Always draw projector lines between FV and TV
⭐ PYQ Analysis
2023
A line AB, 70mm long, is inclined at 30° to HP and 45° to VP. Its end A is 15mm above HP and 20mm in front of VP. Draw its projections and find apparent lengths.
2022
A line PQ 80mm long makes an angle of 40° with VP. Its end P is 20mm above HP and 30mm in front of VP. It is parallel to HP. Draw projections.

Projection of Planes
IMP
▼
Plane Visualization

A plane is a flat surface (like a rectangular card or triangular plate) floating in space. When it lies flat on HP → its FV appears as a line (edge view). When it stands perpendicular to VP → its TV appears as a line. When inclined — both views show the shape but distorted. Only when the plane is PARALLEL to a projection plane do you see its TRUE SHAPE.

Plane PositionFVTV
Parallel to HPLine (edge view)True shape

Perpendicular to bothLineLine
Inclined to HP (⊥ to VP)Line at angle θDistorted shape

✏️ General Method for Inclined Planes
1
Stage 1 — Simple Position
Start with plane in simplest position (parallel to HP). Draw TV as true shape and FV as a line on XY.
2
Stage 2 — Tilt to HP
Rotate FV (line) to make angle θ with XY. Project new TV by drawing horizontal lines from new FV to intersect with vertical projectors from original TV.
3
Stage 3 — Tilt to VP
Rotate TV to make angle φ with XY. Project new FV. This gives final FV and TV for both inclinations.

Types of Solids — Polyhedral & Revolution
IMP
▼
📐 Why This Drawing Matters
Real-World Connection
Every mechanical component is a 3D solid. A bolt is a cylinder + cone. A nut is a hexagonal prism with a hole. Engine pistons, gears, structural columns — all start as solid projections on drawing paper. Mastering this = reading any engineering drawing.
🧠 Solid Types Visual
🔷
Tetrahedron
4 triangular faces
📦
Cube/Cuboid
6 square faces
🏛️
Prism
2 polygonal bases + rectangles
🔺
Pyramid
1 polygon base + triangles meeting apex
⚙️
Cylinder
2 circular ends + curved surface
🍦
Cone
1 circular base + curved surface to apex
🌐
Sphere
Curved surface, all projections are circles
Polyhedral Solids

Bounded by plane surfaces (faces). Examples: Tetrahedron (4F), Cube (6F), Octahedron (8F), Dodecahedron (12F), Icosahedron (20F). All faces are polygons.

Solids of Revolution

Generated by rotating a plane figure about an axis. Cylinder = rotating rectangle. Cone = rotating right triangle. Sphere = rotating semicircle. Torus = rotating circle.

📐
Projections of Solids — Axis Perpendicular
HOT
▼
Mental Model — Axis Understanding

The AXIS of a solid is its spine — the line through its center from base to apex/top. When the axis is perpendicular to HP (standing straight up), look from above → you see the BASE (true shape). Look from front → you see the HEIGHT (rectangle/triangle shape).

When the axis is perpendicular to VP (lying flat pointing at you), look from front → you see the BASE. Look from above → you see the HEIGHT shape.

✏️ Cylinder — Axis ⊥ to HP (Standing)
1
Draw Top View
TV = Circle (diameter = D). This is the true shape of the base as seen from above. Circle represents both top and bottom base (coincide).
2
Draw Front View
FV = Rectangle (width = D, height = H). Two vertical lines = extreme left and right generators. Two horizontal lines = top and bottom circles projected.
3
Side View
SV (if required) = Rectangle same as FV (width = D, height = H). Cylinder looks same from all sides.
✏️ Cone — Axis ⊥ to HP (Standing)
1
Top View
TV = Circle (base). Mark center as apex projection o. Draw circle of diameter D centered at o.
2
Front View
FV = Isosceles triangle. Base = D (horizontal line). Height = H. Apex at top center. Two slant lines = extreme generators.
3
Hidden Lines
In FV, base circle is represented as a thick line at bottom. No hidden lines needed for simple standing cone.
✏️ Pentagonal Prism — Axis ⊥ to HP
1
Top View = True Shape
Draw regular pentagon (side = given). Position it with one edge parallel to VP (nearest to observer) or as specified.
2
Project FV
Project all vertices of pentagon upward. FV = rectangle H units tall. Visible faces show as rectangles, hidden as dashed lines.
3
Mark Edges
Edges visible from front = continuous lines. Edges hidden (behind other faces) = dashed lines. Top and bottom edges = horizontal lines.
🎯 Golden Rule for Solid Projections
  • Axis ⊥ HP → TV shows true base shape, FV shows height
  • Axis ⊥ VP → FV shows true base shape, TV shows height
  • All projections connected by projector lines (thin)
  • Hidden edges always dashed (never skip these)
  • Centerlines always shown (chain line)
⭐ PYQ Analysis

Inclined Solids — Axis Inclined to HP or VP
HOT
▼
Tilted Solid Visualization

Imagine a cylinder standing upright. Now TILT it at 30° toward you. The top view no longer shows a perfect circle — it shows an ellipse! The front view no longer shows a clean rectangle. This is inclined solid projection — it takes the simple position first, then tilts it. Always use the TWO-STAGE method.

1
Stage 1: Simple Position
Draw FV and TV with axis perpendicular to HP (or VP, whichever is simpler). This gives you the simple/true shape projections.
2
Stage 2: Apply Inclination
Tilt the FV to make the required angle with XY. Keep the base on XY. Now re-project the TV by transferring all widths from Stage 1 TV.
3
Transfer Widths
For each point in Stage 2 FV, draw a horizontal projector. For each point in Stage 1 TV, transfer the distance from center line. Where they meet = Stage 2 TV point.
4
Complete Views
Join all projected points. Add hidden lines for invisible edges. Mark center lines. The result shows the solid in its inclined position.
⚠️ Exam Traps
  • Always do Stage 1 first — never jump to inclined position directly
  • Don't forget hidden lines in Stage 2
  • "Base on HP" = base edge/face touching HP in Stage 2
  • Read carefully: "axis inclined to HP" vs "face inclined to HP" — different approaches!
⭐ PYQ
2023
A pentagonal pyramid, base 30mm side and height 65mm, has its axis inclined at 30° to HP. Base is on HP. Draw the projections.

Sections of Solids & True Shape
HOT
▼
🧠 Visual Imagination
The Cutting Plane Mental Movie

Imagine a solid watermelon (or a 3D solid). Now you pass an imaginary knife (cutting plane) through it at an angle. The exposed cross-section = TRUE SHAPE of section. The part you see in the front or top view after cutting = APPARENT SECTION (shows the cut edges and remaining solid). Engineers section solids to reveal internal geometry, like in assembly drawings or manufacturing.

Cutting PlaneShape of Section for CylinderShape for Cone
⊥ to axis (horizontal)CircleCircle
Inclined to axis, cutting all generatorsEllipseEllipse
Parallel to one generatorRectangleParabola
Parallel to axisRectangleHyperbola
Through apex (cone)—Triangle
✏️ Step-by-Step: Section of a Cylinder
1
Draw Both Views
Draw FV (rectangle) and TV (circle) of cylinder in required position. Show cutting plane in FV as a chain thick line (VT-type).
2
Mark Section Points
Divide circle into 12 parts (TV). Project these to FV. Find where each generator meets the cutting plane → these are section points.
3
Project Section Points to TV
From each section point in FV, draw horizontal projectors. In TV, the corresponding generator point gives the TV of section point. Join all = apparent section.
4
True Shape of Section
Draw auxiliary plane parallel to cutting plane. Project all section points perpendicularly onto this auxiliary plane. Distances = distances of generators in TV. Connect → True shape.
✏️ Section of a Cone
1
Set Up Views
Draw standing cone FV (triangle) and TV (circle with generators). Mark cutting plane line in FV.
2
Generator Points
Divide TV circle into 12 generators from apex. Project apex to FV (top of triangle). Mark where cutting plane crosses each generator in FV.
3
TV Points
From each crossing point in FV, project horizontally. In TV, the corresponding generator at same radial position gives the TV point.
4
True Shape
Auxiliary plane method same as cylinder. Result for inclined cut = ellipse. Add section hatching (thin 45° lines) to cut area.
Sectioning Rules
  • Cutting plane = thick chain line (VT type) in view where it appears as edge
  • Section hatching = thin lines at 45°, evenly spaced (3–5mm apart)
  • True shape = auxiliary view drawn on a plane parallel to cutting plane
  • Always show remaining portion of solid with hidden lines where needed
⭐ PYQ
2023

2022

Development of Surfaces
HOT
▼
🧠 Visual Imagination — Paper Unfolding
The Box Unfolding Simulation

Development = cutting and unfolding a 3D solid into a flat sheet. Like taking a cardboard box, cutting the edges, and laying it flat. This flat pattern is the DEVELOPMENT — crucial for sheet metal fabrication (air ducts, tin boxes, funnels, exhaust pipes). Every tin cutter uses developments. You draw it, they cut and fold it into 3D shapes!

Parallel Line Development

Used for prisms and cylinders — solids with parallel lateral edges/generators. Unfold by "rolling" the surface sideways. Length of development = perimeter of base.

Radial Line Development

Used for pyramids and cones — solids with lateral edges/generators meeting at apex. Unfold by rotating surfaces around apex. Slant height = radius of arc.

✏️ Development of Cylinder
1
Calculate Perimeter
Circumference = πD. This is the WIDTH of the rectangular development. Height of rectangle = height of cylinder (H).
2
Draw Rectangle
Draw rectangle of width πD and height H. Divide width into 12 equal parts (each = πD/12, matching 12 generators).
3
Add Cut Line
If cylinder is cut at an angle, plot the cut heights from FV at each generator position. Join smoothly → sinusoidal curve at top.
4
Add Bases
Add two circles (πD diameter) for top and bottom if required. These are the base developments.
✏️ Development of Cone
1
Find Slant Height
Slant height L = √(R² + H²) where R = base radius, H = height. L is the radius of the arc in development.
2
Calculate Arc Angle
Arc angle θ = (R/L) × 360°. This is the angle of the sector (development arc) at apex.
3
Draw Sector
With apex O as center, radius L, draw arc of angle θ. This sector is the lateral surface development of cone.
4
For Cut Cone
Mark slant heights of each generator from FV onto the corresponding arc position. Join to get the cut development profile.
✏️ Development of Prism/Cube
1
Draw Face Sequence
Lay all lateral faces side by side. For a square prism: 4 rectangles, each width = side length, height = H. Total width = 4×side.
2
Add Top and Bottom
Attach base square/polygon on top or bottom of the strip. Add fold lines at each edge junction.
3
Mark Cut Heights
If solid is sectioned, plot the cutting plane intersection heights at each edge from FV. Connect → development profile.
Development Formulas
Cylinder: Width = πD, Height = H Cone: Arc radius = Slant height L, Arc angle = (R/L)×360° Prism: Width = sum of all base edges, Height = H Pyramid: Each triangular face uses TRUE LENGTH of slant edge

Isometric Views — ISO ↔ Orthographic Conversion
HOT

🧠 3D Illusion — Visual Imagination
What is Isometric?

Isometric drawing creates a 3D illusion on flat paper using only three axis directions — all at 120° to each other. It's like looking at a 3D box from a corner. X-axis → 30° right. Y-axis → 30° left. Z-axis → vertical. All measurements along these axes are TRUE LENGTH. Only curved shapes get distorted (use isometric circle method).

Isometric Rules
  • Three axes at 120° to each other
  • Left & right axes at 30° to horizontal
  • Vertical axis = truly vertical
  • No foreshortening — true lengths used
  • Isometric scale factor = 0.816 (if strict)
  • Exam uses full-scale (unitary isometric)
Isometric Circle
  • Circle in ISO = ELLIPSE (rhombus method)
  • Draw isometric square (rhombus)
  • Connect midpoints → 4 arcs join to form ellipse
  • Top face rhombus → horizontal ellipse
  • Front face → tilted ellipse
  • Side face → other tilted ellipse
✏️ Orthographic → Isometric Conversion
1
Identify Dimensions
From orthographic views (FV, TV, SV), extract: Length (L) from FV width, Width (W) from SV or TV depth, Height (H) from FV height.
2
Draw Isometric Box
Start from bottom-front corner. Draw three axes. Plot L along right axis, W along left axis, H vertically. Complete the enclosing isometric box.
3
Transfer Features
Cut away material as indicated by orthographic views. For each step/notch/hole: measure along isometric axes and cut the box accordingly.
4
Add Hidden & Center Lines
In isometric, generally NO hidden lines are shown (unless specifically asked). Add center lines for holes and curves using 30° and 90° directions.
✏️ Isometric → Orthographic Conversion
1
Study Isometric
Identify front, top, and right side faces of the isometric view. Understand all visible features, holes, steps, notches.
2
Draw FV
Looking at the isometric from the front (X-Z plane). Show all visible edges as thick lines, hidden features as dashed. Include all features visible from front.
3
Draw TV
Looking from top (X-Y plane). TV is placed below FV. Project all edges. Features visible from top = solid lines, hidden = dashed.
4
Draw SV
Looking from right side (Y-Z plane). SV placed to right of FV. Project heights from FV, depths from TV (using miter line at 45°).
🎯 Exam Strategy — Isometric
  • Always start with isometric enclosing box (box method)
  • Mark all key dimensions along iso axes FIRST
  • For circles: draw iso square first → 4-arc ellipse
  • For holes: show center lines, then ellipse
  • Iso scale = 1:1 in JNTUK exams (no 0.816 factor)

📐 Why AutoCAD Matters
Industry Reality
100% of professional engineering firms use CAD software. AutoCAD is the global standard. Every drawing you do manually on paper — AutoCAD does it in seconds with PERFECT accuracy. This unit bridges your hand-drawing skills to the digital drafting world.
⚙️ Essential 2D Commands
LINE (L)Draw straight lines. Type L → Enter → click start → click end → Enter to finish.
CIRCLE (C)Draw circles. Type C → Enter → click center → type radius → Enter. Options: 2P (2 points), 3P (3 points), TTR (tangent).
RECTANGLE (REC)Draw rectangles. REC → first corner → type @length,width → Enter.
ELLIPSE (EL)Draw ellipses. EL → axis endpoint → other endpoint → semi-minor axis. Perfect for iso circles.
OFFSET (O)Create parallel copies. O → type distance → select line → click side. Essential for parallel lines.
TRIM (TR)Cut intersecting lines. TR → Enter → click segments to remove. Most-used editing command.
EXTEND (EX)Extend lines to a boundary. EX → select boundary → Enter → click line to extend.
FILLET (F)Round corners. F → R → type radius → Enter → click two lines to fillet.
MIRROR (MI)Reflect objects. MI → select objects → Enter → click mirror line start → end → Yes/No to delete original.
ARRAY (AR)Rectangular or polar array. AR → select → Rectangular/Polar → set rows, cols, angles.
ROTATE (RO)Rotate objects. RO → select → Enter → base point → angle. Positive = CCW, Negative = CW.
SCALE (SC)Scale objects. SC → select → Enter → base point → scale factor (1=same, 2=double, 0.5=half).
HATCH (H)Add section hatching. H → select area → choose pattern (ANSI31 = standard 45° hatching) → scale → OK.
DIMENSION (DIM)Add dimensions. DIMLINEAR for straight, DIMRADIUS for radius, DIMDIAMETER for diameter.

Computer Graphics & Transformations
IMP
▼
Transformation Visualization

Computer graphics manipulates objects using matrix math. Every transformation (move, rotate, scale) is a matrix multiplication behind the scenes. This is the math that powers AutoCAD, Blender, Unity, and every game engine.

⚙️ 2D Transformation Matrices
Translation
New point: x' = x + tx, y' = y + ty Homogeneous: [1 0 tx] [x] [0 1 ty] [y] [0 0 1 ] [1]
Scaling
x' = x × Sx, y' = y × Sy Matrix: [Sx 0 0] [x] [0 Sy 0] [y] [0 0 1] [1] Uniform scaling: Sx = Sy
Rotation (about origin, counterclockwise θ)
x' = x·cosθ - y·sinθ y' = x·sinθ + y·cosθ Matrix: [cosθ -sinθ 0] [x] [sinθ cosθ 0] [y] [0 0 1] [1]
Reflection
About X-axis: x'=x, y'=-y → [1 0 0; 0 -1 0; 0 0 1] About Y-axis: x'=-x, y'=y → [-1 0 0; 0 1 0; 0 0 1] About y=x: x'=y, y'=x → [0 1 0; 1 0 0; 0 0 1]
Composite Transformations

Multiple transformations applied in sequence. Order matters! Rotation then Translation ≠ Translation then Rotation. Multiply matrices in reverse order: T = Tn × ... × T2 × T1

Homogeneous Coordinates

Add extra dimension (w=1) to represent 2D point as [x,y,1]. Enables all transformations (including translation) as matrix multiplication. Standard in all graphics systems.

PCB Graphics
  • PCB = Printed Circuit Board
  • Uses GERBER file format for fabrication
  • Layer-based design (Top copper, Bottom, Silk screen)
  • Design software: KiCad, Eagle, Altium
  • AutoCAD used for mechanical outline of PCB
3D Transformations
  • Same concept + Z axis
  • 4×4 homogeneous matrices
  • Rotation around X, Y, or Z axis
  • Euler angles for combined rotation
  • Used in AutoCAD 3D, CAD/CAM software

⚡ Quick Revision
Transformation Types

Translation (shift), Scaling (resize), Rotation (turn), Reflection (mirror), Shearing (skew). All use 3×3 matrices in 2D with homogeneous coords.

Key Formulas

Rotation: x'=x·cosθ−y·sinθ, y'=x·sinθ+y·cosθ. Scale: x'=Sx·x, y'=Sy·y. Translation: x'=x+tx, y'=y+ty.
