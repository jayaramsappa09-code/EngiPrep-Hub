Concept 1

Binary Search

O(log n) search on a sorted array by repeated halving.

Binary Search Time: T(n) = T(n/2) + O(1)
Concept 2

Merge Sort

Divide and conquer recursive algorithm. Guaranteed O(n log n) time.

Merge Sort Time: T(n) = 2T(n/2) + O(n)
