I

First Order Differential Equations

Linear, Exact & Applications

🔥 Why This Topic Matters

Every changing system in engineering — a cooling engine, a charging capacitor, a growing population — follows a First Order ODE. These equations describe the continuous rate of change of variables. If you want to predict how long a laptop battery takes to discharge, you have to solve a linear ODE!

🧠 Intuition First

An ordinary algebraic equation (e.g., 2x + 3 = 7) says: "Find the number x."

A differential equation (e.g., dy/dx + 2y = 4x) says: "Find the FUNCTION y(x)."

You are reverse-engineering a continuous system based on its slope.

📘 Core Concepts

Linear DE

Has the standard form dy/dx + P(x)y = Q(x). The key is the Integrating Factor (IF).

IF = e^(∫P dx)

Bernoulli's Eq

A non-linear equation that reduces to linear form using substitution. standard form: dy/dx + P(x)y = Q(x)yⁿ

Sub: v = y^(1-n)

Exact DE

Based on partial derivatives. M dx + N dy = 0 is exact if and only if:

∂M/∂y = ∂N/∂x

⚡ Formula Zone

General Solution (Linear DE)

y · (IF) = ∫ Q(x) · (IF) dx + C

Newton's Cooling Law

T(t) = T₀ + (T_i - T₀)e^(-kt)

Solution to Exact DE

∫ M dx (y const) + ∫ N dy (no x) = C

⚠️ Common Mistakes

  • ✗
    Adding +C to IFDon't add +C when finding IF = e^(∫P dx). It cancels out later anyway.
  • ✗
    Exactness Check ConfusionTest is ∂M/∂y = ∂N/∂x, NOT the other way around.

II

Higher Order Linear DE

CF, PI & Variation of Parameters

🧠 System Architecture: CF + PI

Higher order ODEs model complex systems like vibrating beams and electrical LCR circuits. The total response string always equals exactly: Total Response (y) = Transient State (CF) + Steady State (PI).

Complementary Function (CF)

The "Free Response". Determined entirely by solving the homogeneous part f(D)y = 0. Uses roots of the auxiliary equation (real distinct, equal, or complex).

Particular Integral (PI)

The "Forced Response". This is the specific solution generated purely by the forcing function on the RHS f(x). Uses Inverse Operator shortcuts 1/f(D).

⚡ Inverse Operator Shortcuts

Particular Integral engine: PI = [1/f(D)] × X ├── X = e^(ax) │ ├── If f(a) ≠ 0 ➔ PI = e^(ax) / f(a) │ └── If f(a) = 0 ➔ Multiply by x, use f'(a). If f'(a)=0, do x² and f''(a). │ ├── X = sin(ax) or cos(ax) │ ├── Substitute D² = -a² (never sub for D) │ └── If f(-a²) = 0 ➔ multiply x, diff denom wrt D, evaluate. │ ├── X = x^m (Polynomials) │ └── Expand [f(D)]⁻¹ using Binomial Theorem in ascending powers of D │ └── X = e^(ax) * V(x) // Exponential Shift Law └── Shift e^(ax) outside ➔ e^(ax) * [1/f(D+a)] V(x)

⚙️ Variation of Parameters

When PI shortcuts fail (e.g. for tan(x), sec(x), log(x)), use the universal Variation of Parameters sequence.

Step 1: Get CF Solutions

CF = C₁y₁ + C₂y₂

Step 2: Wronskian (W)

W = (y₁)(y₂') - (y₂)(y₁')

Ensure standard form! coefficient of y'' must be 1 to use f(x).

Step 3: PI Formula

PI = -y₁∫(y₂f/W)dx + y₂∫(y₁f/W)dx

III

Partial DE

Surfaces & Multiple Variables

While ODEs describe systems in one variable (like time), PDEs model systems depending on multiple variables, such as heat flow across a 2D surface or sound waves in 3D air.

🔑 Standard Notation

p = ∂z/∂x
q = ∂z/∂y
r = ∂²z/∂x²
t = ∂²z/∂y²
s = ∂²z/∂x∂y
  • Formation of PDE

    Eliminate arbitrary constants (a,b) or arbitrary functions (f,g). Number of arbitrary elements eliminated = order of resulting PDE.

  • Lagrange's Method 95% EXAM PROBABILITY

    For Quasi-linear equation Pp + Qq = R

    dx/P = dy/Q = dz/R

    Find two independent integrals u=c₁ and v=c₂ by solving pairs or using multipliers. General solution: F(u,v) = 0.

IV

Vector Differentiation

The Language of Fields

How do fluid velocities, wind forces, and gravitational pull change from point to point? The Del Operator (∇) unlocks the calculus of 3D spaces.

1. Gradient (Maximum Increase) ∇φ = i(∂φ/∂x) + j(∂φ/∂y) + k(∂φ/∂z) // Result: Vector. Computes Unit Normal vector to surface φ=c
2. Divergence (Sources & Sinks) ∇·F = ∂F₁/∂x + ∂F₂/∂y + ∂F₃/∂z // Result: Scalar. If 0 ➔ Solenoidal vector field (incompressible)
3. Curl (Rotation & Vorticity) ∇×F = Det| i j k | | ∂/∂x ∂/∂y ∂/∂z | | F₁ F₂ F₃ | // Result: Vector. If 0 ➔ Irrotational vector field (conservative)
💥 curl(grad) = 0
∇×(∇φ) = 0
💥 div(curl) = 0
∇·(∇×F) = 0
V

Vector Integration

The Holy Trinity of Theorems

🟢

Green's Theorem

Line ↔ 2D Surface (Plane)

Converts a difficult line integral around a closed curve C into a much simpler double integral over the plane region R bounded by C.

∮_C (M dx + N dy)
= ∬_R (∂N/∂x - ∂M/∂y) dx dy
🌀

Stokes' Theorem

Line ↔ 3D Surface

Upgrades Green's Theorem to 3D. Relates the circulation of a vector field around a boundary curve to the "curl" across the surface spanning it.

∮_C F · dr = ∬_S (∇×F) · n dS
📦

Gauss Divergence

Closed Surface ↔ Volume

Measures the total outward flux of a vector field through a closed surface (like a sphere) by integrating its divergence over the enclosed volume.

∬_S F · n dS = ∭_V (∇·F) dV

⚠ Exam Hack (15 Marks)

If a question asks to "Evaluate by Green's Theorem", you MUST calculate the double integral RHS. If it says "Verify Green's Theorem", you must calculate BOTH LHS (line integral) and RHS (double integral) independently and show they are equal.
