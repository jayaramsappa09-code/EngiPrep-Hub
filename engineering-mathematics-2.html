II

Higher Order Linear DE

CF, PI & Variation of Parameters

Cognitive Memory Tree: Finding PI [ 1/f(D) * X ]

Particular Integral (PI) ├── X = e^(ax) │ ├── If f(a) ≠ 0 ➔ PI = e^(ax) / f(a) │ └── If f(a) = 0 ➔ PI = x * e^(ax) / f'(a) ├── X = sin(ax) or cos(ax) │ ├── Substitute D² = -a² │ └── If f(-a²) = 0 ➔ multiply x, diff denom ├── X = x^m │ └── Expand [f(D)]⁻¹ using Binomial to degree m └── X = e^(ax) * V(x) └── Shift e^(ax) ➔ e^(ax) * [1/f(D+a)] V(x)

IV

Vector Differentiation

Gradient : ∇φ = (i∂/∂x + j∂/∂y + k∂/∂z)φ // Normal to surface
Divergence: ∇·F = ∂F₁/∂x + ∂F₂/∂y + ∂F₃/∂z // If 0 ➔ Solenoidal vector
Curl : ∇×F = Det| i j k | // If 0 ➔ Irrotational vector
V

Vector Integration

High Weightage Theorems

Green's Theorem

Relates line integral around simple closed curve C to double integral over plane region R bounded by C.

∮(M dx + N dy) = ∬(∂N/∂x - ∂M/∂y)dxdy

Stokes' Theorem

Relates surface integral of curl of vector field over surface S to line integral of vector field over its boundary C.

∮F·dr = ∬(∇×F)·n dS

Gauss Divergence

Relates surface integral of vector field over closed surface to volume integral of divergence of region.

∬F·n dS = ∭(∇·F)dV
