рҹ“ҳ Why This Unit Matters в–ј

Every changing system in engineering вҖ” a cooling engine, a charging capacitor, a growing population вҖ” follows a First Order ODE. These equations describe the rate of change of one variable with respect to another.

When your phone battery discharges, the voltage drop follows an ODE. When a hot metal cools in air, the temperature curve is governed by Newton's law вҖ” which IS a first order ODE. When a RL circuit responds to a sudden voltage, the current buildup obeys a linear ODE.

вҡҷпёҸ Engineering Connections
  • Electrical Engineering: RL/RC circuit analysis (Kirchhoff's current law = ODE)
  • Mechanical Engineering: Newton's law of cooling, spring-damper systems
  • Chemical Engineering: Reaction rate equations, diffusion models
  • Civil Engineering: Groundwater seepage, beam deflection
рҹ§  Intuition First вҖ” What IS a Differential Equation? в–ј

An ordinary algebraic equation like 2x + 3 = 7 asks: "Find x."

A differential equation like dy/dx + 2y = 4x asks: "Find a FUNCTION y(x) whose derivative satisfies this relationship."

The Power of "Rate of Change"

Think of y as the temperature of a cup of coffee. dy/dx is how fast the temperature changes. An ODE tells you the law of change вҖ” solving it gives you the actual temperature at any time.

вҡҷпёҸ 1.1 вҖ” Linear First Order ODE в–ј

Standard form: dy/dx + P(x)В·y = Q(x)

The secret weapon: Integrating Factor (IF). Multiply both sides by IF to make the left side a perfect derivative.

Integrating Factor
IF = e^вҲ«P(x)dx
Multiply entire equation by this. LHS becomes d/dx[yВ·IF]
General Solution
y В· IF = вҲ«Q(x)В·IF dx + C
After multiplying by IF, integrate both sides directly
Why it works
d/dx[yВ·e^вҲ«P dx] = QВ·e^вҲ«P dx
Product rule reverse-engineered вҖ” pure genius!

Step-by-Step Procedure

1
Write in Standard Form
Get dy/dx + P(x)y = Q(x). Identify P(x) and Q(x) clearly. If equation is in terms of x as function of y, write dx/dy + Px = Q instead.
2
Find Integrating Factor
Compute IF = e^вҲ«P(x)dx. Do NOT add constant C during this integration вҖ” it cancels anyway.
Example: P = 2/x вҶ’ вҲ«P dx = 2ln|x| вҶ’ IF = xВІ
3
Multiply and Recognize
LHS becomes d/dx[yВ·IF]. This is the magic вҖ” no more product rule mess.
4
Integrate Both Sides
Integrate RHS = вҲ«Q(x)В·IF dx. Add +C. Divide by IF to get y.
вҡҷпёҸ 1.2 вҖ” Bernoulli's Equation в–ј

Standard form: dy/dx + P(x)В·y = Q(x)В·yвҒҝ

This is non-linear because of yвҒҝ. The trick is a substitution that converts it to a linear ODE.

Key Substitution
v = y^(1вҲ’n)
This magical sub eliminates the nonlinearity
Derivative Transform
dv/dx = (1вҲ’n)y^(вҲ’n) dy/dx
Divide Bernoulli equation by yвҒҝ, then substitute v
Resulting Linear Form
dv/dx + (1вҲ’n)PВ·v = (1вҲ’n)Q
Now solve using standard Linear ODE method (IF method)
рҹҡҖ Memory Trick

Bernoulli вҶ’ Banish the nonlinearity: sub v = y^(1вҲ’n). If n=1, equation separable. If n=0, already linear.

Special cases: n=2 вҶ’ Logistic equation (population growth model!)

вҡҷпёҸ 1.3 вҖ” Exact Differential Equations в–ј

Form: M(x,y)dx + N(x,y)dy = 0

Geometric Intuition: An exact equation means there exists a potential function F(x,y) such that dF = 0, so F(x,y) = C is the solution вҖ” a family of level curves!

Exactness Test
вҲӮM/вҲӮy = вҲӮN/вҲӮx
If this holds, the equation is exact. Think: mixed partials of F must be equal (Clairaut's theorem)
Solution: F(x,y)
F = вҲ«M dx + g(y)
Integrate M w.r.t. x. Find g(y) using вҲӮF/вҲӮy = N

Integrating Factors for Non-Exact Equations

IF depends on x only
IF = e^вҲ«f(x)dx
Use when (вҲӮM/вҲӮy вҲ’ вҲӮN/вҲӮx)/N = f(x) only
IF depends on y only
IF = e^вҲ«g(y)dy
Use when (вҲӮN/вҲӮx вҲ’ вҲӮM/вҲӮy)/M = g(y) only
рҹҢЎпёҸ 1.4 вҖ” Newton's Law of Cooling & Growth/Decay в–ј

Newton's Law of Cooling

The rate of cooling is proportional to the temperature difference between the body and surroundings.

ODE Model
dT/dt = вҲ’k(T вҲ’ TвӮҖ)
T = temp of body, TвӮҖ = ambient temp, k = cooling constant
Solution
T(t) = TвӮҖ + (TiвҲ’TвӮҖ)e^(вҲ’kt)
Ti = initial temp. Temperature decays exponentially toward TвӮҖ

Natural Growth and Decay

Growth/Decay Law
dN/dt = kN
k > 0: growth (bacteria), k < 0: decay (radioactive)
Solution
N(t) = NвӮҖ В· e^(kt)
NвӮҖ = initial amount. Half-life: tвӮҒ/вӮӮ = ln2/|k|
RL Circuit
LВ·dI/dt + RI = E(t)
Current buildup in RL circuit вҖ” linear ODE! E = EMF, R = resistance, L = inductance

рҹ“ҳ Why Higher Order ODEs? в–ј

First order ODEs describe systems with ONE storage element (one capacitor, one inductor). But real systems like LCR circuits, vibrating beams, and spring-mass-damper systems have TWO or more energy storage elements вҖ” they need second (or higher) order ODEs.

When you pluck a guitar string, the vibration follows a 2nd order ODE. When your phone vibrates, it's a damped oscillator described by a 2nd order ODE. Higher order = richer physics!

в–¶ Live: Underdamped / Critically Damped / Overdamped Oscillation
рҹ§  General Solution = CF + PI в–ј

For: (aDВІ + bD + c)y = f(x)

Complementary Function (CF)
CF = General soln of f(x)=0
Free response. Contains arbitrary constants CвӮҒ, CвӮӮ. Determined by roots of auxiliary equation.
Particular Integral (PI)
PI = specific soln for f(x)
Forced response. No constants. Uses inverse operator 1/f(D) shortcuts.
General Solution
y = CF + PI
Superposition principle. CF handles homogeneous part, PI handles the forcing.

Finding CF вҖ” Auxiliary Equation Method

Replace D with m: solve amВІ + bm + c = 0

Real & Distinct roots (mвӮҒ вү  mвӮӮ)
CF = CвӮҒe^(mвӮҒx) + CвӮӮe^(mвӮӮx)
Overdamped behavior вҖ” two decaying exponentials
Equal roots (mвӮҒ = mвӮӮ = m)
CF = (CвӮҒ + CвӮӮx)e^(mx)
Critically damped вҖ” fastest decay without oscillation
Complex roots (Оұ Вұ iОІ)
CF = e^(Оұx)[CвӮҒcos(ОІx) + CвӮӮsin(ОІx)]
Underdamped вҖ” oscillating with exponential envelope
вҡЎ Particular Integral вҖ” Shortcut Formulas в–ј
For e^(ax) вҖ” f(D) вү  0
PI = e^(ax) / f(a)
Replace D with a in f(D). If f(a) вү  0, this is the answer.
For e^(ax) вҖ” f(a) = 0 [failure case]
PI = xВ·e^(ax) / f'(a)
Multiply by x and use f'(a). Repeat if f'(a) = 0 too.
For sin(ax) or cos(ax)
Replace DВІ with вҲ’aВІ
PI = sin(ax)/f(вҲ’aВІ) or cos(ax)/f(вҲ’aВІ). If denominator = 0, differentiate denominator.
For xвҒҝ (polynomial)
PI = [f(D)]вҒ»В№ В· xвҒҝ
Expand [f(D)]вҒ»В№ by binomial theorem and operate on xвҒҝ term by term.
For e^(ax)В·V(x)
PI = e^(ax)В·[1/f(D+a)]В·V(x)
Exponential shift theorem вҖ” shift a, then find PI of V(x) with new operator
For xВ·sin(ax) or xВ·cos(ax)
Differentiate PI formula w.r.t. a
Use d/da of the sin/cos PI. Elegant differentiation trick!
вҡҷпёҸ Wronskian & Variation of Parameters в–ј

When PI shortcuts fail (non-standard f(x)), use Variation of Parameters вҖ” the universal method.

Wronskian (Independence Test)
W = |yвӮҒ yвӮӮ |
|yвӮҒ' yвӮӮ'|
W = yвӮҒyвӮӮ' вҲ’ yвӮӮyвӮҒ'. W вү  0 confirms yвӮҒ, yвӮӮ are linearly independent (valid CF)
Variation of Parameters
PI = yвӮҒвҲ«(вҲ’yвӮӮf/W)dx + yвӮӮвҲ«(yвӮҒf/W)dx
yвӮҒ, yвӮӮ from CF. f(x) = RHS. Always works regardless of form of f(x).
1
Find CF вҶ’ yвӮҒ and yвӮӮ
Solve homogeneous equation. Extract two independent solutions yвӮҒ, yвӮӮ from CF.
2
Compute Wronskian W
W = yвӮҒyвӮӮ' вҲ’ yвӮӮyвӮҒ'. Confirm W вү  0.
3
Apply the formula
PI = yвӮҒвҲ«(вҲ’yвӮӮВ·f/W)dx + yвӮӮвҲ«(yвӮҒВ·f/W)dx. No constants in PI вҖ” just the integrals.
вҡЎ LCR Circuits & Simple Harmonic Motion в–ј

LCR Series Circuit

Governing ODE
L dВІq/dtВІ + R dq/dt + q/C = E(t)
q = charge. Equivalent to mass-spring-damper! LвҶ”m, RвҶ”damping, 1/CвҶ”spring constant
In terms of current I = dq/dt
L dI/dt + RI + (1/C)вҲ«I dt = E(t)
Differentiate once more to get 2nd order ODE in I

Simple Harmonic Motion (SHM)

SHM Equation
dВІx/dtВІ + ПүВІx = 0
No damping, no forcing. Pure oscillation. Пү = natural frequency
Solution
x = AВ·cos(Пүt) + BВ·sin(Пүt)
Or: x = RВ·cos(Пүt вҲ’ ПҶ) where R = amplitude, ПҶ = phase
Damped Oscillation
dВІx/dtВІ + 2k dx/dt + ПүВІx = 0
k = damping coeff. Nature of motion depends on kВІ vs ПүВІ
рҹҡҖ LCR вҶ” Mechanical Analogy

L (Inductance) вҶ” Mass m вҖ” resists change in current/velocity

R (Resistance) вҶ” Damping вҖ” dissipates energy

1/C (Capacitance) вҶ” Spring constant вҖ” stores/restores energy

E(t) (EMF) вҶ” Applied force F(t)

рҹ“ҳ Why PDEs? вҖ” The World in Multiple Variables в–ј

An ODE governs how a system changes in ONE variable (time or space). But heat flows through a 2D plate вҖ” temperature T depends on BOTH x, y, and t. Sound waves travel in 3D. These need Partial Differential Equations.

рҹҢҚ Real-World PDE Examples
  • Heat Equation: вҲӮu/вҲӮt = Оұ(вҲӮВІu/вҲӮxВІ) вҖ” temperature distribution in a rod
  • Wave Equation: вҲӮВІu/вҲӮtВІ = cВІ(вҲӮВІu/вҲӮxВІ) вҖ” vibrating string, sound waves
  • Laplace Equation: вҲӮВІu/вҲӮxВІ + вҲӮВІu/вҲӮyВІ = 0 вҖ” steady-state heat, electrostatics
  • Diffusion Equation: same as heat вҖ” pollutant spread in air/water

Geometric view: A PDE solution z = f(x,y) is a SURFACE in 3D space. The PDE constrains how this surface can curve and tilt. The general solution is a family of surfaces.

рҹ§  PDE Notation & Intuition в–ј
Standard Notation
p = вҲӮz/вҲӮx, q = вҲӮz/вҲӮy
p = slope of surface in x-direction. q = slope in y-direction.
2nd Order Notation
r = вҲӮВІz/вҲӮxВІ, s = вҲӮВІz/вҲӮxвҲӮy, t = вҲӮВІz/вҲӮyВІ
r = curvature in x, t = curvature in y, s = twisting
вҡҷпёҸ Formation of PDE в–ј

Method 1: Eliminate Arbitrary Constants

Given a relation with n arbitrary constants вҶ’ differentiate n times to generate n additional equations вҶ’ eliminate all constants.

1
Count constants
e.g., z = (x+a)ВІ + (y+b)ВІ has 2 constants a, b
2
Partial differentiate
вҲӮz/вҲӮx = p = 2(x+a) вҶ’ (x+a) = p/2. вҲӮz/вҲӮy = q = 2(y+b) вҶ’ (y+b) = q/2
3
Substitute back to eliminate constants
z = (p/2)ВІ + (q/2)ВІ вҶ’ 4z = pВІ + qВІ вҶҗ This IS the PDE!

Method 2: Eliminate Arbitrary Functions

Given z = f(ax+by) or z = f(u) + g(v) вҖ” differentiate and eliminate the arbitrary functions f, g.

рҹҡҖ Key Idea

Eliminating 1 arbitrary function вҶ’ 1st order PDE. Eliminating 2 arbitrary functions вҶ’ 2nd order PDE. The number of arbitrary elements = order of resulting PDE.

вҡҷпёҸ Lagrange's Method вҖ” Pp + Qq = R в–ј

The standard 1st order quasi-linear PDE: P(x,y,z)p + Q(x,y,z)q = R(x,y,z)

Geometric Insight: Pp + Qq = R defines a direction field on surface z(x,y). The characteristic curves weave through this field. Two independent integrals of the auxiliary system define the general solution.

Auxiliary (Lagrange) System
dx/P = dy/Q = dz/R
These are the characteristic equations. Solve to find two independent integrals u(x,y,z) and v(x,y,z).
General Solution
F(u, v) = 0 or u = f(v)
Where u = cвӮҒ and v = cвӮӮ are two independent integrals of the auxiliary system
1
Write Lagrange system
dx/P = dy/Q = dz/R вҖ” identify P, Q, R from the PDE
2
Find 1st integral
Use pairs: dx/P = dy/Q вҶ’ solve вҶ’ u(x,y,z) = cвӮҒ
3
Find 2nd integral
Use another pair or multiplier method вҶ’ v(x,y,z) = cвӮӮ
4
Write general solution
F(u, v) = 0 or ПҶ(cвӮҒ, cвӮӮ) = 0 is the general solution
рҹҡҖ Multiplier Method Trick

If no direct integration works, try multipliers l, m, n such that: lP + mQ + nR = 0 AND l dx + m dy + n dz = 0 is an exact differential. Then integrate to get one integral directly.

вҡҷпёҸ Homogeneous Linear PDE with Constant Coefficients в–ј

Form: (aD^n + bD^(nвҲ’1)D' + ... + kD'^n)z = f(x,y) where D = вҲӮ/вҲӮx, D' = вҲӮ/вҲӮy

Method is analogous to higher-order ODEs! Replace D вҶ’ m, D' вҶ’ 1 (or 1, m) to get auxiliary equation.

CF (Homogeneous part)
CF = ПҶвӮҒ(y+mвӮҒx) + ПҶвӮӮ(y+mвӮӮx)
m values from auxiliary eq. ПҶбөў are arbitrary functions. For equal roots: ПҶвӮҒ(y+mx) + xВ·ПҶвӮӮ(y+mx)
PI for e^(ax+by)
PI = e^(ax+by) / f(a,b)
Substitute DвҶ’a, D'вҶ’b in operator f(D,D'). Failure rule applies similarly.
PI for sin/cos(ax+by)
Replace DВІвҶ’вҲ’aВІ, DD'вҶ’вҲ’ab, D'ВІвҶ’вҲ’bВІ
Only valid for 2nd order terms. If result = 0, use general method.

рҹ“ҳ Why Vector Differentiation? The Language of Fields в–ј

Physics is full of fields: temperature at every point in a room, wind velocity at every point in the atmosphere, electric field around a charge. These fields need vector calculus to describe how they change, flow, and rotate.

вҡҷпёҸ Engineering Applications
  • Gradient: Find the direction of steepest temperature rise вҶ’ where heat flows fastest
  • Divergence: Detect sources and sinks in fluid flow вҶ’ does fluid compress or expand here?
  • Curl: Detect rotation in velocity fields вҶ’ vorticity in turbulence, magnetic fields
  • Directional Derivative: Rate of change in any chosen direction

рҹ“Ҳ Gradient вҖ” Direction of Maximum Increase в–ј

Intuition: Imagine you're on a hilly landscape. The gradient at your feet points in the direction you'd walk to climb FASTEST. Its magnitude tells you how steep that climb is.

Gradient Formula
вҲҮПҶ = (вҲӮПҶ/вҲӮx)i + (вҲӮПҶ/вҲӮy)j + (вҲӮПҶ/вҲӮz)k
вҲӮПҶ/вҲӮx = rate of change in x, вҲӮПҶ/вҲӮy in y, вҲӮПҶ/вҲӮz in z. Together they form the steepest ascent vector.
Directional Derivative
D_Гў ПҶ = вҲҮПҶ В· Гў
Rate of change of ПҶ in direction Гў (unit vector). Maximum when Гў is parallel to вҲҮПҶ. Zero when Гў вҠҘ вҲҮПҶ (along level surface).
Unit Normal to Surface
nМӮ = вҲҮF / |вҲҮF|
For surface F(x,y,z) = c, the gradient вҲҮF is ALWAYS perpendicular to the surface. Normalize to get unit normal.
Maximum Directional Derivative
Max D_Гў ПҶ = |вҲҮПҶ|
The maximum rate of change equals the magnitude of gradient. Occurs in the gradient direction itself.
рҹ“Ҳ Divergence вҖ” Source or Sink? в–ј

Intuition: Place a tiny balloon in a fluid flow field F. If the fluid expands the balloon вҶ’ div F > 0 (source). If it compresses вҶ’ div F < 0 (sink). If no change вҶ’ div F = 0 (solenoidal вҖ” no sources or sinks).

Divergence Formula
div F = вҲӮFвӮҒ/вҲӮx + вҲӮFвӮӮ/вҲӮy + вҲӮFвӮғ/вҲӮz
For F = FвӮҒi + FвӮӮj + FвӮғk. Result is a SCALAR. Measures net outflow per unit volume.
Solenoidal Field
div F = вҲҮВ·F = 0
Incompressible fluid flow, magnetic field B вҖ” no net sources. What flows in = what flows out.
Laplacian
вҲҮВІПҶ = div(grad ПҶ) = вҲӮВІПҶ/вҲӮxВІ + вҲӮВІПҶ/вҲӮyВІ + вҲӮВІПҶ/вҲӮzВІ
вҲҮВІПҶ = 0 вҶ’ Laplace equation (harmonic functions вҖ” steady state heat, electrostatics)
рҹ“Ҳ Curl вҖ” Rotation in the Field в–ј

Intuition: Drop a tiny paddle wheel into a fluid flow F. If the wheel spins вҶ’ curl F вү  0 (rotational). If it doesn't spin вҶ’ curl F = 0 (irrotational). The direction of curl F is the axis of rotation; magnitude = speed of spin.

curl F = вҲҮГ—F = |i j k |
|вҲӮ/вҲӮx вҲӮ/вҲӮy вҲӮ/вҲӮz|
|FвӮҒ FвӮӮ FвӮғ |
Determinant expansion. Result is a VECTOR. Each component gives rotation rate about that axis.
Irrotational Field

Conservative force field (gravity, electrostatics). Line integral is path-independent. F = вҲҮПҶ for some scalar ПҶ.

рҹ“ҳ Why Vector Integration? вҖ” Measuring Flow & Work в–ј

Vector differentiation told us how fields change locally. Vector integration tells us the cumulative effect вҖ” total work done by a force, total fluid flux through a surface, total charge enclosed in a volume.

рҹҢҚ Engineering Applications
  • Line Integral: Work done by a force field along a path (W = вҲ«FВ·dr)
  • Surface Integral (Flux): Total fluid flow through a surface per unit time
  • Green's Theorem: Converts line integral вҶ” double integral (2D version of Stokes)
  • Stokes' Theorem: Converts surface integral вҶ” line integral (circulation = curl flux)
  • Divergence Theorem: Converts volume integral вҶ” surface integral (total outflow)
вҡҷпёҸ Line Integrals вҖ” Work & Circulation в–ј

Intuition: Walk along a curved path in a force field. At each tiny step dr, the work done = FВ·dr (dot product = force component along motion Г— step length). Total work = sum of all these tiny contributions = line integral.

Line Integral (Work)
W = вҲ«_C FВ·dr = вҲ«_C (FвӮҒdx + FвӮӮdy + FвӮғdz)
F = force field. dr = displacement. Parametrize the curve C to evaluate.
Circulation
Circulation = вҲ®_C FВ·dr
Closed loop line integral. Non-zero вҶ’ rotational field. For irrotational: circulation = 0.
Conservative Field Test
вҲ«_C FВ·dr independent of path вҹә curl F = 0
If irrotational AND simply connected, F = вҲҮПҶ. Then вҲ«FВ·dr = ПҶ(B) вҲ’ ПҶ(A). No parametrization needed!
Scalar Potential
F = вҲҮПҶ вҶ’ ПҶ = вҲ«FвӮҒdx + g(y,z)
Find ПҶ by integrating. Work = ПҶ(end) вҲ’ ПҶ(start). Gravity, electrostatics have scalar potentials.

Parametric Evaluation

1
Parametrize the curve C
Express x, y, z as functions of t: x=x(t), y=y(t), z=z(t), t: aвҶ’b
2
Compute dr
dx = x'(t)dt, dy = y'(t)dt. Substitute into the integrand.
3
Evaluate the 1D integral
Everything is now in t. Integrate from a to b.
вҡҷпёҸ Surface Integrals вҖ” Flux Through a Surface в–ј

Intuition: Hold a net in a river. The flux = total volume of water passing through the net per second. Net flow depends on: water speed, net area, AND the angle between flow and net (cos Оё factor = dot product with normal).

Flux through Surface S
Flux = вҲ¬_S FВ·nМӮ dS = вҲ¬_S FВ·dS
nМӮ = outward unit normal. dS = nМӮ dS = vector area element.
Evaluation Formula
вҲ¬_S FВ·nМӮ dS = вҲ¬_R FВ·nМӮ |1/cosОі| dA
Project onto coordinate plane. cosОі = nМӮВ·kМӮ (for projection on xy-plane). |вҲҮF|/|вҲӮF/вҲӮz| often used.
For surface z=f(x,y)
dS = вҲҡ(1+(вҲӮz/вҲӮx)ВІ+(вҲӮz/вҲӮy)ВІ) dA
Area scaling factor. nМӮ = (вҲ’вҲӮz/вҲӮx i вҲ’ вҲӮz/вҲӮy j + k) / |...|
вҡЎ Green's Theorem вҖ” 2D Circulation = Enclosed Curl в–ј

Intuition: The total circulation (rotation) around a closed boundary = the sum of all the local rotations inside. Like adding up all the tiny paddle wheels inside a region вҖ” their total spin equals the big circulation around the edge.

Green's Theorem
вҲ®_C (M dx + N dy) = вҲ¬_R (вҲӮN/вҲӮx вҲ’ вҲӮM/вҲӮy) dA
C = boundary of region R (traversed counterclockwise). Converts line integral вҶ” double integral. Also used to find AREA: A = ВҪвҲ®(x dy вҲ’ y dx)
рҹҡҖ Memory Aid

Green = 2D version of Stokes. вҲӮN/вҲӮx вҲ’ вҲӮM/вҲӮy is the "2D curl" (z-component of вҲҮГ—F where F = Mi + Nj).

Area by Green's: A = вҲ®_C x dy = вҲ’вҲ®_C y dx = ВҪвҲ®_C (x dy вҲ’ y dx)

вҡЎ Stokes' Theorem вҖ” Surface Curl = Boundary Circulation в–ј

Intuition: The total circulation of a vector field around the boundary of a surface = the total curl (rotation) through that surface. Green's theorem is the 2D special case of Stokes' theorem!

Stokes' Theorem
вҲ®_C FВ·dr = вҲ¬_S (вҲҮГ—F)В·dS = вҲ¬_S (вҲҮГ—F)В·nМӮ dS
C = boundary curve of surface S. F = vector field. вҲҮГ—F = curl of F. The circulation around the rim = flux of curl through the surface.
1
Compute curl F = вҲҮГ—F
Find all three components of curl F using the determinant formula.
2
Find unit normal nМӮ of surface S
For plane or curved surface, find nМӮ (outward or using right-hand rule with boundary C).
3
Evaluate (вҲҮГ—F)В·nМӮ
Dot product, then integrate over the surface projection.
вҡЎ Gauss Divergence Theorem вҖ” Volume Div = Surface Flux в–ј

Intuition: The total fluid flowing OUT through a closed surface = the total "creation" of fluid (divergence) inside the volume. Count all sources inside вҶ’ matches total outward flow. No fluid created = closed surface = net zero flux (solenoidal).

Gauss Divergence Theorem
вҲҜ_S FВ·nМӮ dS = вҲӯ_V (вҲҮВ·F) dV
S = closed surface bounding volume V. вҲҮВ·F = divergence. Converts surface flux integral вҶ” volume integral. Often much easier to evaluate one or the other!
рҹҡҖ The Big Three вҖ” Comparison

Green's Theorem (2D): вҲ®C вҶ’ вҲ¬R вҖ” boundary to area

Stokes' Theorem (3D): вҲ®C вҶ’ вҲ¬S вҖ” boundary curve to surface (curl)

Divergence Theorem (3D): вҲҜS вҶ’ вҲӯV вҖ” surface to volume (divergence)

All three say: "Integral over boundary = integral of derivative over region." It's the same idea at different dimensions!
